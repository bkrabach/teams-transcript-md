# Icons

Drop PNG icons here named:

- `icon-16.png` (16×16)
- `icon-32.png` (32×32)
- `icon-48.png` (48×48)
- `icon-128.png` (128×128)

The extension manifest references these paths. They are optional — if
absent, Edge/Chrome use a generic puzzle-piece icon and the extension
still works.

To remove the references entirely, delete the `"icons"` block from
`manifest.json`.
