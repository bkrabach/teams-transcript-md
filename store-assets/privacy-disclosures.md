# Privacy practices (Chrome Web Store)

Chrome's privacy form is a cascade of checkboxes plus a single
free-text justification. These answers match what the extension
actually does.

## Question 1 — Are you collecting or using user data?

**Yes**

## Question 2 — Which categories of data?

Tick **only** this one:

- [x] **Website content**

Leave **No** on all of these:

- [ ] Personally identifiable information
- [ ] Health information
- [ ] Financial and payment information
- [ ] Authentication information
- [ ] Personal communications
- [ ] Location
- [ ] Web history
- [ ] User activity

## Question 3 — Justification for "Website content"

```
Reads the current tab's transcript pane or SharePoint Stream API response only when the user clicks Capture. The content is processed locally and delivered to the user as a download. No copy is sent anywhere else.
```

## Question 4 — Certifications (tick all three)

- [x] I do not sell or transfer user data to third parties, apart from the approved use cases
- [x] I do not use or transfer user data for purposes that are unrelated to my item's single purpose
- [x] I do not use or transfer user data to determine creditworthiness or for lending purposes
